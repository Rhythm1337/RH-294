# Selecting Hosts with Host Patterns
# Example
## 1.Making The Playbook
**ansible.cfg**
```
inventory = inventory
```

**inventory**
```
[rhel]
server[a:b]

[fedora]
server[c:d]

[webservers]
servera
serverc

[dbservers]
serverb
serverd
```
## 2. Listing Inventory Hosts
ansible --list all

**output**
```
servera
serverb
serverc
serverd
```
